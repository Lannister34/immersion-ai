# Roadmap

## Current Features

- **Characters** — PNG V2 cards: list, view, create, edit, delete, AI wizard, drag & drop / paste avatars
- **Chat** — JSONL format, streaming via KoboldCpp, edit/delete all messages, auto-titles
- **Lorebooks** — list, view, edit, delete, AI generator
- **Scenarios** — AI-generated RP scenarios with preview before save
- **LLM Server** — built-in KoboldCpp management (start/stop, model selection)
- **Settings** — KoboldCpp connection, user settings, sampler parameters

## Planned

### High Priority

- **User persona** — configurable user name, description, avatar for RP context
- **Instruct / Context templates** — ChatML, Alpaca, Llama3, Mistral, custom template editor
- **Token counter** — tokenizer integration for prompt budget tracking
- **Chat export** — save conversations as JSON, TXT, or Markdown

### Medium Priority

- **Multi-provider support** — OpenAI, Anthropic, Ollama, OpenRouter, TabbyAPI
- **Preset system** — named generation parameter presets (temperature, top_p, top_k, rep_pen, etc.)
- **Group chats** — multi-character conversations
- **Import / Export** — CharX, JSON, YAML character formats; bulk import

### Lower Priority

- **Chat branching** — message swipes, alternative response trees
- **Themes & UI customization** — custom color schemes, font sizes, layout options
- **Chat backups** — automatic versioned backups of conversations
- **Regex scripts** — find/replace rules for generation output processing
